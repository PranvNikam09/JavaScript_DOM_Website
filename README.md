# SkyBound - Flight Booking & Management System

SkyBound is a modern, responsive flight booking web application developed entirely using **HTML**, **CSS**, and **JavaScript (DOM Manipulation)**. This project demonstrates the power of dynamic UI creation, where the entire frontend structure and functionality are built programmatically using JavaScript.

## 🚀 Key Features

- **Dynamic UI Rendering**: All UI components are created and managed via JavaScript DOM methods, showcasing a deep understanding of browser APIs.
- **Modern Responsive Design**: Built with a "Mobile First" approach using Bootstrap 5, ensuring a seamless experience across all devices.
- **Interactive Navbar**: Features a sleek, responsive navigation bar with an offcanvas mobile menu for better accessibility.
- **Flight Ticket Generation**: Includes a dedicated module for generating flight tickets, handling user interactions and data presentation.
- **Vibrant Visuals**: Utilizes modern iconography from Bootstrap Icons and Font Awesome, along with premium imagery to create a wowed user experience.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3, Bootstrap 5
- **Icons**: [Bootstrap Icons](https://icons.getbootstrap.com/), [Font Awesome](https://fontawesome.com/)
- **Architecture**: Dynamic DOM Manipulation

## 📂 Project Structure

```text
├── images/             # Visual assets and icons
├── navbar.js           # Logic for the responsive navigation system
├── script.js (1-10)    # Modular scripts handling different page sections
├── ticket.js           # Core logic for ticket generation
├── index.html          # Main entry point (Root container)
├── ticket.html         # Ticket display template
├── style.css           # Global stylesheets and custom animations
└── ticket.css          # Specific styles for the ticket module
```

## 🌟 How It Works

The application leverages a central container in `index.html` which is then populated by various JavaScript modules (`script.js`, `navbar.js`, etc.). This approach minimizes the initial HTML payload and allows for highly modular and maintainable code structure.
